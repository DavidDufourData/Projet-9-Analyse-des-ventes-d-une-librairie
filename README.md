# Analyse des ventes d’une librairie

## Contexte et objectifs

L’objectif est d’étudier le comportement d’achat des clients, analyser l’évolution des ventes et établir une prévision simple des ventes futures.

## Soft Skills
- Vulgarisation des résultats statistiques  
- Esprit d’analyse orienté business  
- Interprétation rigoureuse des tendances  
- Formulation de recommandations claires  

## Compétences
- Analyses temporelles (tendance, saisonnalité)  
- Tests statistiques qualitatifs et quantitatifs sur des données pour tester et valider des hypothèses (Khi-2, T  de Student, ANOVA...)  
- Prévisions (modèles simples)  
- Construction de visualisations analytiques  

## Outils utilisés
- Python  
- Pandas, Matplotlib, Seaborn, Statsmodels
